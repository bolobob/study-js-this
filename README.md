設問５（まとめ）
======

* つまり`this`は何によって変わるでしょうか？
* 以下の`this`は何を指すでしょうか？

```js
fn0();
function fn0() {
  fn1();
  function fn1() {
    fn2();
    function fn2() {
      console.log(this);
    };
  };
};
```
